Programar automáticamente (Windows)

Ejecuta el scraper cada mañana automáticamente con el Programador de tareas de Windows.

# En PowerShell como Administrador:
$action = New-ScheduledTaskAction -Execute "python" \`
  -Argument "C:\ruta\ai-notes-agent-v4\news_scraper.py"
$trigger = New-ScheduledTaskTrigger -Daily -At "08:00"
Register-ScheduledTask -TaskName "AINewsScraper" \`
  -Action $action -Trigger $trigger -RunLevel Highest
